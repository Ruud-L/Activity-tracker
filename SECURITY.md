# Security Policy

## Official download source

Download builds only from official project sources:

- This GitHub Pages site (`./Activity_Tracker_Setup.exe`)
- The official GitHub Releases page (`https://github.com/<GITHUB_USER>/<REPO>/releases/latest`)

## Payment safety

- This page never asks for your PayPal password.
- Use only the official PayPal URL configured by the project owner.

## URL and HTTPS checks

- Always verify the domain before downloading or donating.
- Ensure the browser shows `https://` in the address bar.
- Do not trust copied links from unknown sources.
